Dicas de html

HTML é a linguagem básica de qualquer página na internet. Utilizando seus elementos de linguagem, as Tags, é possível mudar a cor e o tamanho do texto, criar tabelas,
adicionar imagens e manipular vários detalhes na formatação. Na Wikipédia essa linguagem é manipulada para facilitar o trabalho dos editores, modificando o modo como
se adiciona imagens, cria tabela, cria seções, entre outras funções que estão no Guia de edição, e também para proibir algumas tags para que os editores não cometam excessos
na edição dos artigos. Vou colocar alguns exemlos aqui:

Tags de formatação

< b >< /b > Deixa o texto em negrito < b >texto em negrito < / b > texto em negrito
< i >< /i > Deixa o texto em itálico < i >texto em itálico < / i > texto em itálico
< u >< /u > Deixa o texto sublinhado < u >texto sublinhado < / u > texto sublinhado

Obs: Retirar os espaços das tags com os sinais<>
